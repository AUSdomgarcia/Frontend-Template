# Nuworks Front-End Template


## Features

*SASS Compilation - auto compile sass
*UnCSS - remove unused css
*Optimize image
*Minify CSS
*Minify JS


## Getting Started

- Install: `npm install`
- Run: `grunt` - To compile SASS and watch scss changes
- Run `grunt build` for building dist copy
